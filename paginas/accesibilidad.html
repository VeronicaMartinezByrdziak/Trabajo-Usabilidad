DECLARACIÓN DE ACCESIBILIDAD

Compromiso con la accesibilidad

En Medline, nos comprometemos a garantizar la accesibilidad a todas las personas, incluyendo a aquellas con discapacidades visuales, motoras o cognitivas.

Características de accesibilidad implementadas

  • 🧭 Navegación con teclado: Puedes moverte entre enlaces y formularios usando la tecla Tab.
  • 🖼️ Textos alternativos: Todas las imágenes importantes contienen el atributo alt para ser interpretadas por lectores de pantalla.
  • 🌐 Identificación del idioma: El idioma principal está definido como lang="es" en cada página.
  • 🔤 Etiquetado semántico: El código HTML está estructurado semánticamente para facilitar la lectura por dispositivos de asistencia.
  • 🔗 Enlaces accesibles: Todos los enlaces tienen texto significativo que describe su destino.
  • 📱 Diseño adaptativo: La web se adapta a diferentes dispositivos mediante media queries y estructura flexible con Flexbox.
  • 🎨 Contraste adecuado: El diseño cumple con los niveles mínimos de contraste según la norma WCAG 2.1.
  • 🎯 Enlace de salto: Existe un enlace visible con tabulador para saltar directamente al contenido principal.

Nuevas funcionalidades de accesibilidad

  • 🌙 Estilo modo noche
  • 🖨️ Estilo específico para impresión
  • 🎨 Estilo de alto contraste
  • 🔠 Estilo con fuente de mayor tamaño
  • 🔳 Estilo combinado (contraste + letra grande)

Compatibilidad con formatos accesibles

Los documentos electrónicos ofrecidos desde esta web están disponibles en formatos accesibles como PDF etiquetado y Word con estilos estructurados. Esto facilita la lectura mediante tecnologías de asistencia.

Cómo mejorar la accesibilidad

Si encuentras barreras de accesibilidad o deseas hacernos llegar alguna sugerencia, puedes escribirnos a través del formulario de contacto.
